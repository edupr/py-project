# Game Project

Para correr el juego, debes seguir las siguientes instrucciones en la terminal:

```sh
cd game
python3 main.py
```

# Charts Project

Para correr la gráfica, debes seguir las siguientes instrucciones en la terminal:

```sh
cd charts
python3 main.py
```

Y la gráfica generada estará en `pie.png`
